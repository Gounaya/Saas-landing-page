# SaaS Landing Page

## Description
This landing page is designed to promote our SaaS (Software as a Service). It showcases key features, benefits, and client testimonials, while providing a simple and effective way for users to sign up or request a demo.

## Features
- **Responsive Design**: Compatible with all devices (desktops, tablets, smartphones).
- **Testimonials Section**: Displays reviews and feedback from our satisfied clients.
- **Sign-Up Form**: Allows users to easily sign up for our service.
- **Social Media Integration**: Links to our social media profiles.
- **SEO Optimized**: Enhances the page's visibility on search engines.

## Installation
To install and run this landing page locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Gounaya/Saas-landing-page.git
   ```
2. Navigate to the project directory:
   ```bash
   cd saas-landing-page
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Usage
Once the development server is running, open your browser and go to `http://localhost:5173` to see the landing page in action.
